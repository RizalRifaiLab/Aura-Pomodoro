## How They Work:

### Complete Sound (`complete.mp3`)
- **Triggers when:** A Pomodoro session or break reaches 0:00
- **Use case:** Signals that it's time to switch to the next phase
- **Volume:** 50% (adjustable in code)

### Running Sound (`running.mp3`)
- **Triggers when:** You click the Play button to start the timer
- **Use case:** Audio feedback that the timer has started
- **Volume:** 50% (adjustable in code)

### Pause Sound (`pause.mp3`)
- **Triggers when:** You click the Pause button while the timer is running
- **Use case:** Audio feedback that the timer has been paused
- **Volume:** 50% (adjustable in code)

## Sound Settings

You can control all sounds through the Settings panel:
1. Click the gear icon (top-right)
2. Toggle "Sound Effects" on/off
3. All three sounds will respect this setting

## Customization

To replace any sound:
1. Find a new sound file (1-3 seconds recommended)
2. Convert to MP3 format if needed
3. Replace the existing file with the same name
4. Keep the same filename (`complete.mp3`, `running.mp3`, or `pause.mp3`)
5. Refresh the browser to hear the new sound

## Recommended Sound Characteristics:
- **Duration:** 1-3 seconds (short and pleasant)
- **Volume:** Medium (app will set to 50%)
- **Type:** Gentle tones, chimes, or notification sounds
- **Format:** MP3 (best browser compatibility)
